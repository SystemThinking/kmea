# Onboarding pre študentov (SK)

## 1. Klikni na assignment
- Dostaneš link od učiteľa.
- Prihlás sa správnym GitHub účtom.

## 2. Over meno
- Použi školské označenie (napr. ST-001).

## 3. GitHub ti vytvorí repozitár
- Automaticky vznikne súkromná kópia.

## 4. Klonuj repozitár
```bash
git clone <url>
```
# Ukážka z reálnych aktivít


## 5. Prvý commit
- Doplň *AboutMe* a pushni.

## 6. Sync PR
- Učiteľ pošle PR cez sync assignment.
- Tvoj PR otvor → resolve conflicts → merge.
