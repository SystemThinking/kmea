# Onboarding pre učiteľov (SK)

## 1. Vytvorenie assignmentu
- Priprav si template repozitár.
- Otvor GitHub Classroom → Create Assignment.
- Vyber template.
- Zadaj prefix v tvare `2025_ST-`.
- Student dostane repozitár automaticky.

## 2. Prefixy a mená
- Prefix musí byť konzistentný: `2025_ST-001`, `2025_ST-002`, ...
- Ak študent zvolí nesprávny účet, nevidí assignment.
- POZOR- PREFIX sa nastavuje cez malé písmená pod názvom projektu. Ak si nedáš pozor, neuvidíš.

## 3. Ako funguje workflow
- Učiteľ vytvorí assignment.
- Študent klikne → GitHub Classroom vytvorí repozitár.
- Každý repozitár je súkromný.
- Sync assignment aktualizuje iba existujúce repozitáre.

## 4. Sync assignmentu (učiteľský pohľad)
- Vždy robíš sync z Classroom → template → študenti.
- Sync vytvorí PR do študentských repozitárov.
- Študent PR vyrieši a merge-ne.

## 5. Správa triedy
- Môžeš premenovať repozitáre.
- Môžeš pridávať asistenta.
- Môžeš pridávať ďalšie assignmenty.
