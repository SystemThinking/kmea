# Inštalácia nástrojov na macOS

## Potrebné nástroje

- **Homebrew**
- **Git**
- **Node.js LTS**
- **VS Code** (odporúčané)

## Inštalácia Homebrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Git

```bash
brew install git
```

## Node.js LTS

```bash
brew install node
```
