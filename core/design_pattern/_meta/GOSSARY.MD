# Glossary (Design Patterns)

- **DAO** – typ entity (napr. `knife`, `q12`, `7ds`).
- **FM-Core** – jednotná hlavička (front matter); nemodifikujeme ju na mieru per typ, len dopĺňame user header.
- **User Header** – voliteľný doplnok nad FM-Core (bez menenia jadra).
- **SSOT** – Single Source of Truth (tu: FM-Core + patterns).
