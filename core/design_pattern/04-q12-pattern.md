# Pattern – Q12

- **DAO:** `q12`
- **Rozcestník sekcie:** `content/docs/<locale>/q12/index.md` (analogicky ku KNIFE/7Ds)
- **Pozor:** `body/` je len v **template** (oddelenie header/body), nie vo finálnych cestách obsahu.
- **FM-Core:** rovnaký ako inde; doplnky idú cez user header.

Princíp navigácie a linkovania – **rovnaký** ako KNIFE (relatívne odkazy, späť na Hub, stabilný slug ak treba).
